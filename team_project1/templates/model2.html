펄서 판별

펄스 신호 정보를 통해 해당 신호가 중성자 별에서 보내온 신호인지 아닌지를 예측해드립니다.

딥러닝
  • 은닉층 수: 6 + 출력층 + 입력층 = 8
  • 활성화 함수 : relu 사용 (성능이 가장 높게 측정됨)
  • 학습률 : 0.003
  • batch_size : 64
  • 얼리스탑핑 적용: val_accuracy 기준으로 적용(epoch=1000)
  • 배치 정규화, 드랍아웃 사용 - 성능 상승(속도) + 과적합 방지
  • 타겟의 불균형을 보정하기 위해 class weight 파라미터 사용
  • 추천 epoch 수 - 10~20
  • 성능은 Test-F1 score 기준 0.7559 입니다.
    • 딥러닝 GITHUB

    머신러닝
  • LogisticRegression 모델을 사용
  • StandardScaler() 적용
  • GridSearch를 통해 최적의 하이퍼파라미터 발견
  • 성능은 Test-F1 score 기준 0.8940 입니다.
    • 머신러닝 GITHUB
